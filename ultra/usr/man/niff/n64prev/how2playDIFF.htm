プレビューワーの操作方法

共通操作方法

  • 十字ボタンとＳＴＡＲＴボタンで、メニューの操作を行います。
  • Ａボタンでアニメーションを再生します。
  • Ｂボタンでアニメーションを逆再生します。
操作方法として、以下の２つの方法を採り入れています。

１.トラックボール

表示オブジェクトを仮想的なトラックボールの中に投影しているとして、 トラックボールを回転させるようにオブジェクトを回転させ、あらゆる角度から 見ることが出来ます。
  • ３Ｄスティック＋Ｚトリガで、トラックボール操作をエミュレートしていま す。Ｚトリガ押下で、ボールに触れている状態、そのまま３Ｄスティックを 操作するとモデルが回転します。
  • ３Ｄスティックを倒したまま、Ｚトリガを離すと、モデルは回転を続けます。
  • Ｒボタンを押しながら、３Ｄスティックを上下で、カメラとモデルの距離を 遠ざけたり、近付けたり出来ます。
  • Ｃボタンユニットの４ボタンは、モデルのＸ、Ｙの並行移動に使用します。 加速度移動するので気をつけて下さい。
  • Ｒボタンを押しながら、Ｃボタンユニットの４ボタンで、表示位置の 微調整が出来ます。
  • Ｌボタンを押しながら、３Ｄスティックを倒すと、Ｘ、Ｙのスケーリングの 調整が出来ます。
  • Ｌ＋Ｒボタンを押しながら、３Ｄスティックを倒すと、Ｚのスケーリングの 調整が出来ます。

    ２.ヘリコブタークルーズ

    カメラを仮想的なヘリコプターに見立て、オブジェクトの周囲を移動して、 見て回ります。
  • ３Ｄスティックを操縦桿と見立てて、方向を制御します。 横方向の傾倒でＸＺ平面の方向を変更します。多少カメラの傾きにも 影響します。
  • 縦方向の傾倒量でカメラ方向を天頂から真下までの範囲を動かします。
  • Ｌ、Ｒボタンで、それぞれカメラ方向へ前進、後退です。
  • 十時ボタンの上下で、移動のギヤ比を変更します。上でシフトアップ、 下でシフトダウンです。全部で６速あり、始めは３速です。
  • Ｃボタンユニットで、カメラ方向に対しての垂直上昇、降下と左右の 並行移動です。 上記２パターンの操作方法は、メニューによって切替えることが出来ます。

    メニュー操作方法

    最初の状態では、メニューは表示されていません。 STARTボタンを押すとメニューが表示されます。 現在の項目は、赤で表示され、それ以外の項目は、白で表示されます。
    • 十字ボタンの上下で項目を移動します。
    • 数値を設定する項目では、十字ボタンの左で減少、右で増加させます。
    • STARTボタンで項目を選択します。
    最初のメニューは以下のような構成になっています。
    • OBJECT
    • CAMERA
    • CONTROL
    • LIGHT
    • FOG
    • BG COLOR
    • ANIMATION
    • VIDEO
    • EXIT
      EXIT 項目を選択すると、メニューを消去します。
    [OBJECT メニュー] OBJECT 項目を選択すると OBJECT メニューが表示されます。 OBJECT メニューは以下のような構成になっています。
    • SCENE/NINTENDO
      SCENE または NINTENDO がトグルに切り替わる項目です。 SCENE のときは、シーンを表示し、NINTENDO のときは、N64ロゴが表示され ます。
    • RESET
      オブジェクトの状態をリセットします。SCENE オブジェクトが選択され、オ ブジェクトの位置や向きやスケールが初期状態に戻ります。
    • EXIT
      OBJECT メニューを消去します。
    [CAMERA メニュー]
    • CONTROL/SCENE
      CONTROL または SCENE がトグルに切り替わる項目です。 SCENE のときは、シーンのカメラを(利用できれば)利用し、CONTROL のとき は、コントローラで操作できるカメラを利用します。
    • RESET
      カメラの状態をリセットします。CONTROL カメラが選択され、カメラの位置 や向きやスケールが初期状態に戻ります。
    • EXIT
      CAMERA メニューを消去します。
    [CONTROL メニュー]
    • CAMERA
      コントローラでカメラを動かします。
    • OBJECT
      コントローラでオブジェクトを動かします。
    • LIGHT1
      コントローラで光源1を動かします。
    • LIGHT2
      コントローラで光源2を動かします。
    • LIGHT3
      コントローラで光源3を動かします。
    • LIGHT4
      コントローラで光源4を動かします。
    • LIGHT5
      コントローラで光源5を動かします。
    • LIGHT6
      コントローラで光源6を動かします。
    • LIGHT7
      コントローラで光源7を動かします。
    • TRACKBALL/CRUISE
      コントローラのモードを切り替えます。
    • EXIT
      CONTROL メニューを消去します。
    [LIGHT メニュー]
    • DEFAULT/SCENE
      DEFAULT では n64prev が設定している光源を利用します。 SCENE では、シーンの光源設定が利用されるので、光源1〜7は無効です。
    • AMBIENT
      環境光を設定します。
    • LIGHT1 COLOR
      光源1の色を設定します。
    • LIGHT2 COLOR
      光源2の色を設定します。
    • LIGHT3 COLOR
      光源3の色を設定します。
    • LIGHT4 COLOR
      光源4の色を設定します。
    • LIGHT5 COLOR
      光源5の色を設定します。
    • LIGHT6 COLOR
      光源6の色を設定します。
    • LIGHT7 COLOR
      光源7の色を設定します。
    • RESET
      すべての光源の色と向きを初期設定に戻します。
    • EXIT
      LIGHT メニューを消去します。
    [FOG メニュー]
    • OFF/ON
      Fog の ON/OFF を切り替えます。
    • R, G, B, A
      Fog の色を指定します。
    • NEAR, FAR
      Fog のかかる距離を指定します。
    • RESET
      Fog パラメータを初期設定に戻します。
    • EXIT
      FOG メニューを消去します。
    [BG COLOR メニュー]
    • R, G, B
      背景色を指定します。
    • RESET
      背景色を初期設定に戻します。
    • EXIT
      BG COLOR メニューを消去します。
    [ANIMATION メニュー]
    • TOGGLE/STEP
      Ａボタン、Ｂボタンの挙動を決定します。 TOGGLE モードでは、ボタンでアニメーション再生／停止を切り替えます。 STEP モードでは、ボタンでアニメーションを1フレーム進めたり、戻したり します。
    • EXIT
      BG COLOR メニューを消去します。
    [VIDEO メニュー]
    • GAMMA OFF/ON
      ガンマ補正の ON/OFF を決定します。
    • GAMMA DITHER OFF/ON
      ガンマディザの ON/OFF を決定します。
    • DIVOT OFF/ON
      ディボットの ON/OFF を決定します。
    • DITHER FILTER OFF/ON
      ディザフィルタの ON/OFF を決定します。
    • EXIT
      VIDEO メニューを消去します。
