nb2nt

Converts NIFF binary to text

Syntax

nb2nt input-file

Explanation

Outputs the NIFF data input-file in text format to standard output.

By entering the command as

nb2nt input-file > output-file
the output can be dumped to output-file.

Notes

If there are errors in the NIFF data, sometimes it cannot be output as text.
