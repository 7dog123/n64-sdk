Texture Node Properties

The following tabs are available in the Texture node properties:

  • Node name
  • Type
  • Wrapping

Node name

The "Node name" tab sets the name of the Texture node.

The node name is displayed if the node is linked to the Name area. If it is not linked, then "Unknown" is displayed here. To change the node name, edit the node name that is displayed here. If you want to set a new node name for a node that is not linked to the Name area, simply delete "Unknown" and input the desired name. Conversely, if you want to delete the node name, simply delete the name that is displayed here.

The characters that can be used for node names are the ASCII characters "!" (0x21) to "~" (0x7e). Note that the name cannot contain a space nor any Japanese characters.

Type

Use the "Type" tab to specify the texture format and bit size.

For "Format" select one of the following items for texture format:

  • [I]
  • [IA]
  • [CI]
  • [YUV]
  • [RGB]
  • [RGBA]

For "Bit size" select one of the following items for the bit size of each pixel of the texture.

  • 4 bits
  • 8 bits
  • 16 bits
  • 32 bits

The selectable bit sizes will change automatically, depending on the format that has been selected.

Wrapping

Use the "Wrapping" tab to specify texture wrapping.

Texture wrapping is specified for both the S direction and the T direction. Select from among the below items for each specification:

  • Wrap
  • Clamp
  • Mirror
