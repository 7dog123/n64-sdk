Animation Node Properties

The following tabs are available in the Animation node properties:

  • Node name
  • Type
  • Frame rate
  • Loop

Node name

The "Node name" tab sets the name of the Animation node.

The node name is displayed if the node is linked to the Name area. If it is not linked, then "Unknown" is displayed here. To change the node name, edit the node name that is displayed here. If you want to set a new node name for a node that is not linked to the Name area, simply delete "Unknown" and input the desired name. Conversely, if you want to delete the node name, simply delete the name that is displayed here.

The characters that can be used for node names are the ASCII characters "!" (0x21) to "~" (0x7e). Note that the name cannot contain a space nor any Japanese characters.

Type

Use the "Type" tab to check the animation type setting.

The animation type setting cannot be changed.

Frame rate

Use the "Frame rate" tab to check the frame rate setting of the animation.

The animation frame rate setting cannot be changed.

Loop

Use the "Loop" tab to set the number of times to loop the animation.

To make this setting, check the "Set number of animation loops" box, and then enter a decimal value to specify the number of animation loops.
