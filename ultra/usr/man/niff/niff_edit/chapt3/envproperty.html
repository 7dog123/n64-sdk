Environment Node Properties

The following tabs are available in the Environment Node Properties dialog:

  • Node name
  • Fog
  • Frame buffer
  • Z buffer

Node name

Use the "Node name" tab to set the name of the Environment node.

The node name is displayed if the node is linked to the Name area. If it is not linked, then "Unknown" is displayed here. To change the node name, edit the node name that is displayed here. If you want to set a new node name for a node that is not linked to the Name area, simply delete "Unknown" and input the desired name. Conversely, if you want to delete the node name, simply delete the name that is displayed here.

The characters that can be used for node names are the ASCII characters "!" (0x21) to "~" (0x7e). Note that the name cannot contain a space nor any Japanese characters.

Fog

Use the "Fog tab to make fog-related settings.

Check the "Use Fog" box to enable the use of fog. When the box is not checked, this function is disabled. This tab can also be used to specify the fog color, the fog starting point, and the fog saturation point.

When fog is enabled, the following fog-related items can be set:

  • A hexadecimal value in 0xRRGGBBAA format can be specified for fog color.
  • A decimal number from 0 to 100 can be specified for the fog starting point.
  • A decimal number from 0 to 1000 can be specified for the fog saturation point.

Frame buffer

Use the "Frame buffer" tab to clear the frame buffer.

When you want to clear the frame buffer, put a check in the Clear Frame Buffer box. When this box is checked, you can also specify a hexadecimal value in 0x00RRGGBB format for the FILL color when clearing.

Z buffer

Use the "Z buffer" tab to clear the Z buffer.

When you want to clear the Z buffer, put a check in the Clear Z Buffer box. When this box is checked, you can also specify a hexadecimal value in 0x00RRGGBB format for the FILL color when clearing.
