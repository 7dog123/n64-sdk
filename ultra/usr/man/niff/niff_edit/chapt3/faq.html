FAQ

Below is a FAQ concerning the use of NIFF Editor.

How do I update to the latest version of NIFF Editor?

NIFF Editor support is available from the NTSC-ONLINE home page. Here you can find answers to NIFF Editor questions and download the latest software version.

I updated NIFF Editor but nothing seems to have changed...

After the NINTENDO64 Developer's Kit has been installed, the first time you update NIFF Editor you need to manually go in and delete all files and sub-directories from the directory shown below:

  • c:\nintendo\n64kit\niff\bin\niffedit (this is the default location)

In addition, if you are using the Update setup program and the only component that is already installed is NIFF Editor, then you must first go in and delete NIFF Editor using Add/Remove Programs in the Control Panel.

How do I change the home page address displayed in Online Support in the Help menu?

The Online Support home page address is set with the following Registry key:

  • HKEY_CURRENT_USER\Software\Intelligent Systems\NIFF Editor\Settings\OnlineAddress

By changing this key, the user can specify any address they want. Before doing this, please backup the existing Registry and take special care, since in the worst-case scenario the changes can lead to faulty operation of NIFF Editor and even Windows itself.

I can't preview when I select Quick Preview from the Tools menu.

When Quick Preview is selected, NIFF Editor executes the quickprev.bat file. This batch file describes a process for IS-VIEWER64 to load the NIFF file without first performing makemask. This works because version 1.03p and later of nload.exe contains a function to perform makemask, obviating the need to perform makemask ahead of time. Thus, if you are using an earlier version of nlaod.exe, you will not be able to preview the file because it will be loaded into IS-VIEWER64 without performing makemask.

If you have a version of nload.exe earlier than version 1.03p, add the makemask process to the quickprev.bat file, or download the latest version of nload.exe from Intelligent Systems' IS-VIEWER64 home page (http://www.intsys.co.jp/tool/isv/).
