Revision History

The current version of NIFF Editor is not the final version. Development will continue in order to expand the functions of the system even further. Information will be added to the manual as well as to this Revision History page every time the version is updated, providing the user smooth access to the new functions. At the same time, new versions will continue to be put out in order to provide the user with essential functions and a user-friendly environment. Please review this page every time a new version is released.

This Revision History describes all functions added and bugs fixed up to the time of the release of the current version of NIFF Editor.
