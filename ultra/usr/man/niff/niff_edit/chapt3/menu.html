Menus

The following menu is displayed at the top of the NIFF Editor window:

When these menu items are selected, the following sub menus are displayed:

File menu Edit menu View menu Tools menu Help menu

File menu

The following commands are available in the File menu:

New... Create a new NIFF file.
Open... Open a NIFF file in NIFF Editor. When this command is selected, the File dialog appears, enabling you to select a file or input the name of a NIFF file to open.
Save Save the current contents of NIFF Editor by overwriting to the file with the same file name.
Save As... Save the current contents of NIFF Editor to the file with the specified name. When this command is selected, the File dialog appears, enabling the user to select a file or input a file name to save the NIFF file.
Properties Display the dialog window to view node names and flag information.
Recent files Display the names of the 10 most recent files opened by NIFF Editor. Nothing is displayed if no files have been opened since NIFF Editor was set up.
Exit Quit out of NIFF Editor.

Edit menu

The following commands are available in the Edit menu:

Cut Cut the node and copy it to the clipboard.
Copy Copy the node to the clipboard.
Paste Paste the node that has been copied to the clipboard.
Delete Delete the selected node. At this time, the node's links are deleted, and then if other nodes do not link to it, the node itself is deleted. After that, the node's child nodes are also deleted, providing no nodes other than the parent node link to them.
Change node name Change the name of the node in the window.
To change the node name, edit the node name that is displayed in the dialog window. If you want to set a new node name for a node that is not linked to the Name area, simply delete "Unknown" and input the desired name. Conversely, if you want to delete a node name, simply delete the name that is displayed.
The characters that can be used for node names are the ASCII characters "!" (0x21) to "~" (0x7e). Note that the name cannot contain a space nor any Japanese characters.

View menu

The following commands are available in the View menu:

[Tool bar] Show/hide the tool bar
[Status bar] Show/hide the status bar
Split Move the split bar to divide the NIFF Editor window into left and right halves. When this menu item is selected, the mouse cursor automatically moves to the top of the split bar, making it easy to move the split bar.
Large icons Show large icons in the list displayed on the right side of the NIFF Editor window.
Small icons Show small icons in the list displayed on the right side of the NIFF Editor window.
List Display the list as a simple list on the left side of the NIFF Editor window.
Details Include information about each item in the list displayed on the left side of the NIFF Editor window.

Tools menu

The following commands are available in the Tools menu:

[Convert] Convert the loaded NIFF file into a display list.
[Preview] Convert and compile the loaded NIFF file and transfer the created file to IS-VIEWER64 for previewing.
[Quick Preview] Send the loaded NIFF file to IS-VIEWER64 for previewing without compiling.
[Options...] Specify the name for the file to be used for converting and previewing.

Help menu

The following commands are available in the Help menu:

Topic search Display NIFF Editor Help. The Help files are in HTML format, so when this command is selected, the browser is activated and NIFF Editor Help is displayed.
[Online support] The NIFF Editor online support home page NTSC-ONLINE is displayed in the browser.
About NIFF Editor Display NIFF Editor version information.
