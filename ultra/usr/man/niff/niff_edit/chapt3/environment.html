Operating Environment

In order to use NIFF Editor, the computer must fully provide the operating environment described below. Please confirm that these conditions are fully met before setting up NIFF Editor.

Item Description
Operating System Microsoft Corp.'s Windows 95, Windows 98, Windows NT Server Version 4.0 (IBM PC/AT compatible version), or Windows NT Workstation Version 4.0 (IBM PC/AT compatible version).
Memory At least 32Mbytes recommended.
Free Disk Space Requires at least 10Mbytes of free space.
Browser A browser is needed to display NIFF Checker Help and to access online support. We recommend using Microsoft Corp.'s Internet Explorer 3.0 or later, or Netscape Communications Corp.'s Netscape Navigator 3.0 or later or Netscape Communicator 4.0 or later. Please set up one of these browsers ahead of time and check that they are working properly.
SCSI Port Required for Preview and Quick Preview. If running under Windows NT Version 4.0, operation can only be guaranteed for the AHA-2940U from Adaptec. Other SCSI ports have not been checked so their proper operation cannot be guaranteed.
IS-VIEWER64 Required for Preview and Quick Preview. Please set up IS-VIEWER64 and confirm that it is working properly ahead of time.
Quick Preview requires nload.exe from IS-VIEWER64 version 1.03p or later. However, if makemask is added to quickprev.bat then an earlier version of nload.exe can also be used. For details, see Quick Preview in Setup.
exeGCC Used to convert and preview a NIFF file. exeGCC is not required if only Quick Preview will be used to preview NIFF files.

If the operating environment is prepared, refer to Setup and begin setting up NIFF Editor.
