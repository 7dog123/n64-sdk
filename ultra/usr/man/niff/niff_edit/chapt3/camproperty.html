Camera Node Properties

The following tabs are available in the Camera node properties:

  • Node name
  • Type
  • Clipping plane
  • Others

Node name

The "Node name" tab sets the name of the Camera node.

The node name is displayed if the node is linked to the Name area. If it is not linked, then "Unknown" is displayed here. To change the node name, edit the node name that is displayed here. If you want to set a new node name for a node that is not linked to the Name area, simply delete "Unknown" and input the desired name. Conversely, if you want to delete the node name, simply delete the name that is displayed here.

The characters that can be used for node names are the ASCII characters "!" (0x21) to "~" (0x7e). Note that the name cannot contain a space nor any Japanese characters.

Type

The "Type" tab sets the camera type.

For camera type, you can choose either "Perspective camera" or "Ortho camera."

Clipping plane

The "Clipping plane" tab sets the camera's clipping planes.

The following elements can be set in the clipping plane tab.

  • Near clipping plane
  • Far clipping plane
  • Right clipping plane
  • Left clipping plane
  • Top clipping plane
  • Bottom clipping plane

Please use floating decimal values to set the clipping planes.

Others

Use the "Others" tab to set the Fovy value, the Aspect ratio and the Scale value.

Please use floating decimal values to set the Fovy value, the Aspect ratio and the Scale value.
