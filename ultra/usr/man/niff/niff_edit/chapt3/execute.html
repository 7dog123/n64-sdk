Starting & Quitting

Here we explain the procedures for starting and quitting NIFF Editor.

Starting

When NIFF Editor is properly installed using the procedure described in Install, a "NIFF Editor" menu item is added from the Start button to Programs Å®[n64kit]. NIFF Editor can be activated by selecting the "NIFF Editor" command from this menu item's submenu.

or, by following the below procedure you can start up NIFF Editor and at the same time read a NIFF file.

  • Use Explorer to double-click on a file with a "nif" or a "niff" extension, or click with the right mouse button to display the pop-up menu and then select "Open."
  • Drag & Drop a NIFF file icon on the NIFF Editor icon (or an icon shortcut) on the desktop or in Explorer.

Quitting

Use any of the procedures listed below to quit NIFF Checker.

  • Select Exit from the File menu.
  • Press X on the keyboard while holding down the Atl key.
  • Click the left-side icon on the Title Bar and select Close.
  • Double-click the left-side icon on the Title Bar.
  • Click on the Close button on the right side of the Title Bar.

If a NIFF file has been updated when you act to quit NIFF Editor, a message will appear asking whether the changes should be saved.

Select Yes to save the changes. Select No to not save the changes. Select Cancel if you do not want to quit NIFF Editor.
