Chapter 2　NIFFeditor for SGI



(7) convert_options Files

　The NIFF editor for SGI can call up the niff2gfx command from a menu. This is a cnverter for converting NIFF data into N64 graphics data. THe niff2gfx command has a variety of switches for conversion purposes. Setting these switches will allow the output of graphics data under the required settings. (For details, see the chapter on niff2gfx.)

　With NIFF editor, these niff2gfx option switches can be set by overwriting the converter_options file. The .niffcnvops file in the home directory is the file for such settings.

　These can be set using the [Convert to N64 DL]->[Set Options and Go] menu, but it can also be edited by simply using a text editor. This file is loaded when NIFF editor starts.

　Contents of convert_options File　
../usr_data
default.c
-header default.h
-label default
-scale
-niffinfo
-nocomdef
-regmask
-vtxshare
-vtxcache
-strictflat

This list shows the contents of the text file for this file.

The first line indicates the directory from which the file will be output. This must be specified.

The second line specifies the file name that will output the DL. This also must be specified.

The option flags(-****) must be added to the third line and those that follow. The sequence is optional. The option flags are not required, but if they are not inserted, niff2gfx is executed without any options. Consequently, those options that are required by niff2gfx must be noted. For a description of options required by niff2gfx.

For detail about the option, please refer to the chapter on niff2gfx.
