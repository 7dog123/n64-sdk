3. NIFFeditor Tour

Features of NIFFeditor for SGI

You can do the following using NIFFeditor for the SGI.

  • Load NIFF data and display inter-node links in tree format
  • Modify data contents

    • Change links
    • Change the names of various nodes
    • Change the data (flags, tags) of various nodes

  • Save modified data in files as NIFF data
  • Convert from NIFF data to N64 data (display list, NVF)
  • Use the N64 Emulator Board or IS-Viewer64 to preview on N64 machines

From Loading to Previewing Files

Use the following process to load a file and preview it. Various options are provided along the way for editing your data.

1. Open NIFF data files

Select a file from the menu by choosing Open from the File menu. When you click OK, the NIFF data will be loaded and a tree will be displayed. The square buttons with labels, such as obj_0, are each assigned what are called nodes in NIFF. The arrow connected to that node is its link. If other data is already being displayed, before opening (loading) a file, clear the display by choosing New from the File menu. Then load (open) the file you want.

2. View data

You can view the contents of each node by double-clicking the node. The current settings will be displayed in the window (called a dialog box) that appears when you double-click. You can change these settings.

3. Modify data

Choose EditMode from the Edit menu on the main menu. If the buttons in the EditMode menu are yellow, data can be modified. To reflect the changes in the dialog box in the actual NIFF data, click the OK or Apply button. Click Cancel to leave the data unchanged. Links can also be changed.

4. Save data

Data can be saved in a file as NIFF data by choosing Save or Save As from the File menu. Save will write the data to the loaded file, and Save As allows you to specify a different file.

5. Preview

You can view data using an N64 Control Deck by choosing Nintendo64 preview (Emu-Board) or Nintendo64 preview (IS) from the N64tool menu.
