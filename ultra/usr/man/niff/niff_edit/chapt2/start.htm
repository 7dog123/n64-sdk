NIFFeditor for SGI

2. Startup Method

When Nintendo 64 NIFF/NuSystem is installed on an SGI machine, the following directory structure will be created:

/usr/local/n64kit/niff
The NIFF-related execution files will all be stored in the following directory: 
/usr/local/n64kit/niff/bin/
Consequently, a path should be connected in advance to this directory. Next, type the following command and press Enter: 
%cp /usr/local/n64kit/niff/bin/.niffcnvops   ~/
With this command, the conversion options file will be copied to the user's /home directory. The execution script for NIFFeditor for SGI is indicated below. 
/usr/local/n64kit/niff/bin/ne2
To start NIFFeditor for the SGI, type the following and press Enter: 
ne2
