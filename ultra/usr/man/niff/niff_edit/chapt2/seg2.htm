(3) Menus
　The following menus are available in NIFF editor.
  • Main Menu

    　The above figure shows the main menu. The operations performed by the various items are described below.

    [File]−[New] Command

    　This command clears the tree display in the main window. Because data cannot be properly read from a new file while a tree is currently displayed, be sure to load the file only after executing [New] command.

    [File]-[Open] Command

    　This command loads a new file and displays the tree thereof. You can selec the file to be loaded from a dialog box or enter its name.

    Figure　Load File Dialog Box

    [File]-[Save] Command

    　This command saves (overwrites) the loaded file. When you are asked "Overwrite OK," click the [OK] button to save (overwrite) the file. If you do not wish the existing file to be overwriten, click the [Cancel] button.

    Figure　Overwrite and Save Warning Dialog box

    [File]-[Save As ...] Command

    　This command saves the data under a new file name. As with the [Open] command, you enter the file name in a File Specification dialog box.

    [Edit]-[Edit Mode] Command

    　When this is ON(when the ON mark is colored), it is possible to modify (add, delete) links, and change node names and node flag settings.This is a toggle setting.

    [Edit]-[Edit Flags] Command

    　Currently not supported.

    [View] Command

    　Currently not supported.

    [N64 tool]-[Nintendo64 Preview(IS)] Command

    　Use the IS-Viewer64 to perform a Preview on the NINTENDO64 Control Deck.

    [N64 tool]-[Nintendo64 Preview(Emu-Board)] Command

    　This allows you to perform a Preview on the NINTENDO64 Control Deck using an Emulator Board.

    [N64 tool]-[Convert to N64 DL]-[Set Options and Go] Command

    　This command starts the converter for outputting NINTENDO64 graphics data. You can set the converter options.

    　The options that have been set are reflected in the .niffcnvops file stored in the user's /home directory, and are valid until the next conversion.

    [N64 tool]-[Convert to N64 DL]-[Set Output Directory] Command

    　This command allows you to specify the directory where you want to store NINTENDO64 graphics data output. When the dialog box for specifying the output destination is displayed, select the desired directory. This directory specification is also saved in the .niffcnvops file.

    [N64 tool]-[Convert to OMDB] Command

    　Outputs daatabase object manager of NINTENDO64. Currently not supported.

    [N64 tool]-[Kill [gn]load]

    　This cancels the gload (for the Emulator Board) and nload(for the IS-Viewer64) process performed during preview.

    [HELP]-[Product Information] Command

    　This command displays the NIFF editor version information.
