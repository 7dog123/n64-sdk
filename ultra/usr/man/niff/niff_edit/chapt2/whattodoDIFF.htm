NIFFeditor for SGI

3. NIFFeditorを使ってみよう

NIFFeditor for SGIでできること

NIFFeditor for SGI では次のことができます。
  • NIFFデータを読み込んでノード間のリンクをツリー形式に表示する。

  • データの中身の変更

    • リンクを変更
    • 各ノードの名前の変更
    • 各ノードのデータ(フラグなど)の変更

      →制限事項

  • 変更したデータをNIFFデータとしてファイルにセーブ。

  • Converting NIFF data to N64 data (display list, NVF)

  • N64エミュレータボードまたはIS-VIEWER64を使って、N64実機でプレビュー

ファイルの読み込みからプレビューまで

  1. NIFFデータファイルを読み込む。

    メニューから[File]-[Open]でファイルを選択します。

    [OK]をクリックすると、NIFFデータが読み込まれてツリーが表示されます。

    obj_0などのラベルがついた四角いボタンがそれぞれNIFFでいうノードにあたります。 そのノードを結んでいる矢印がリンクです。

  2. データを見る。

    ノードをマウスの左ボタンでダブルクリックすると、各ノードの中身を見ることができます。

    ダブルクリックで出たウィンドウ(ダイアログとよぶ)上では現在の設定内容が表示されていますが、これを変更することによって設定を変えることができます。

  3. データを変更する。

    メニューから[Edit]−[EditMode]を選んで下さい。 EditModeのメニューにあるボタンが黄色になっていれば、データの変更が可能です。

    ダイアログで変更した内容を実際のNIFFデータに反映させるには[OK]または[Apply]ボタンをクリックして下さい。[Cancel]をクリックするとデータは変更されません。

    リンクを変更することもできます。

  4. データをセーブする。

    メニューから[File]-[Save]または[Save As ...]でNIFFデータとしてファイルにセーブすることができます。

    [Save]の場合は読み込んだファイルに上書きされます。
    [Save As ...]の場合は別のファイルを指定することができます。

  5. プレビューする。

    When [N64tool]-[Nitendo64 preview(Emu-Board)], [Nitendo64 preview(IS)] are selected from the menu, it is possible to preview with N64.
