(2) Startup Method

　When "Nintendo64 Developers Kit" is installed on SGI machine, the following directory structure will be created:

/usr/local/n64kit/niff

　The NIFF-related execution files will all be stored in the following directory.

/usr/local/n64kit/niff/bin/

　Consequently, a path should be connected in advance to this directory.

　Next, type the following command and press enter.

%cp /usr/local/n64kit/niff/bin/.niffcnvops [space] ~/ [enter]

　With this, the option setup files for N64 Preview will be copied to the home directory.

　The execution script for NIFF editor for SGI is indicated below.

/usr/local/n64kit/niff/bin/ne

　To start NIFF editor for SGI, type following and press enter.

ne[enter]
