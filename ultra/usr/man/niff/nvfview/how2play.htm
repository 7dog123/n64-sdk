Operating the nvfview Previewer

Button Operation

ButtonAction
Z + Control StickRotates the object.
Control Stick (UP)Decreases the distance from the camera to the object.
Control Stick (DOWN)Increases the distance from the camera to the object.
C TOP, BOTTOM, LEFT, RIGHTMoves the object upward, downward, to the left, to the right.
Note: If this operation is performed when the SCENE camera is selected, Previewer will switch to the CONTROL camera automatically.
LMoves the Animation back 1 frame.
RMoves the Animation forward 1 frame.
APlays Animation. (Toggle action)
BPlays Animation backward. (Toggle action)
Control PadSelects the menu.
STARTDetermines the menu.

Manipulating the Menus

In the initial state, the menu is not displayed. The menu appears when the START button is pressed. The current item is displayed in red, and the other items are displayed in white.
  • Use the Control Pad UP/DOWN to move among items.
  • For items that involve the setting of numeric values, use the Control Pad LEFT to decrease the value, and RIGHT to increase it.
  • Use the START button to select an item.
The Main Menu contains the following items:
  • OBJECT
  • CAMERA
  • FOG
  • BG COLOR
  • ANIM PITCH
  • VIDEO
  • ICON
  • EXIT
    Select EXIT to close out the menu.

[OBJECT Menu]

When the OBJECT item is selected, the OBJECT Menu is displayed. The OBJECT Menu contains the following items:
  • SCENE/NINTENDO
    This item is used to toggle between SCENE and NINTENDO. SCENE displays the scene, while NINTENDO displays the N64 logo.
  • RESET
    Resets the state of the object. The SCENE object is selected, resetting the object's position, direction and scale to their initial states.
  • EXIT
    Exits from the OBJECT menu.

[CAMERA Menu]

  • CONTROL/SCENE
    This item is used to toggle between CONTROL and SCENE. When SCENE, the scene camera is used (if it can be used). When CONTROL, the camera that can be manipulated by the Controller is used.
  • RESET
    Resets the camera state. The CONTROL camera is selected, resetting the camera's position, direction and scale to their initial states.
  • EXIT
    Exits from the CAMERA menu.

[FOG Menu]

  • OFF/ON
    Switches Fog ON/OFF.
  • R, G, B, A
    Specifies the Fog color.
  • NEAR, FAR
    Specifies the Fog distance.
  • RESET
    Resets the Fog parameters to their initial settings.
  • EXIT
    Exits from the FOG menu.

[BG COLOR Menu]

  • R, G, B
    Specifies the background color.
  • RESET
    Resets the background color to the initial setting.
  • EXIT
    Exits from the BG COLOR menu.

[ANIM PITCH Menu]

  • Playback speed (1, 0.5, 0.25, 2)
    Switches the animation playback speed. This speed does not apply when the L/R buttons are used to move through frames.
  • EXIT
    Exits from the ANIM PITCH menu.

[VIDEO Menu]

  • GAMMA OFF/ON
    Determines whether gamma compensation is ON/OFF.
  • GAMMA DITHER OFF/ON
    Determines whether gamma dither is ON/OFF.
  • DIVOT OFF/ON
    Determines whether divot is ON/OFF.
  • DITHER FILTER OFF/ON
    Determines whether dither filter is ON/OFF.
  • SCREEN RESOLUTION
    Switches the screen resolution. If the selected resolution cannot be used, 320x240 16bpp is used.
  • EXIT
    Exits from the VIDEO menu.

[ICON Menu]

  • OFF/ON
    Makes and displays icons for inverse kinematics Joint and Effector.
  • EXIT
    Exits from the ICON menu.
