5. CamNode

(1) Cam Node Description (English)

u32 external_lookat_obj_file_name_index When the Obj linked to the Cam look-at point references an external file, that file name is specified with an ExternalNameList index number. In this case, please specify cam_lookat_obj = ExternalNameList index number.

Please do not include the directory name etc. when specifying this file name.

If there is no external reference, please set it to 0xffffffff.

u32 external_lookat_obj_name_index Uses an ExternalNameList index number to describe the external file reference Obj name.

In this case, please specify cam_lookat_obj = ExternalNameList index number.

If there is no external reference, please set it to 0xffffffff.

u32 external_eye_obj_file_name_index When the Obj linked to the Cam viewpoint references an external file, that file name is specified with an ExternalNameList index number.

Please do not include the directory name etc. when specifying this file name.

In this case, please specify cam_eye_obj = ExternalNameList index number.

If there is no external reference, please set it to 0xffffffff.

u32 external_eye_obj_name_index Uses an ExternalNameList index number to describe the external file reference Obj name.

In this case, please specify cam_eye_obj = ExternalNameList index number.

If there is no external reference, please set it to 0xffffffff.

u32 external_up_obj_file_name_index When the Obj linked to the upward Cam references an external file, that file name is specified with an ExternalNameList index number.

Please do not include the directory name etc. when specifying this file name.

In this case, please specify cam_up_obj = ExternalNameList index number.

If there is no external reference, please set it to 0xffffffff.

u32 external_up_obj_name_index Uses an ExternalNameList index number to describe the external file reference Obj name.

In this case, please specify cam_up_obj = ExternalNameList index number.

If there is no external reference, please set it to 0xffffffff.
