NIFF 1.0 Specification

3. Binary Format

3.10 StList

1) StList Area

The StList area describes the s,t coordinate of a texture. The StList area is mapped as follows.
StListHeader u32 st_list_tag
u32 st_list_header_size
u32 st_list_size
u32 st_group_num
u32 nintendo_extension_block_size
u32 user_extension_block_size
u32 st_group_size(0)
...
u32 st_group_size(st_group_num - 1)
StGroup(0)
...
StGroup(st_group_num - 1)
NintendoExtensionBlock
UserExtensionBlock

StList Area Memory Map
u32 st_list_tag Tag indicating StList area. 
TAG_ST_LIST     0x00070000
u32 st_list_header_size Size of StListHeader area.
u32 st_list_size Size of StList area.
u32 st_group_num Total number of StGroup area.
u32 nintendo_extension_block_size Nintendo extension block size.
u32 user_extension_block_size User extension block size.
u32 st_group_size Size of each StGroup area.
StGroup 'S' and 'T' value per 1 shape.
NintendoExtensionBlock Nintendo extension block.
UserExtensionBlock User extension block.
