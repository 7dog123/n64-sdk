u32 nintendo_extension_block_size If an extension data is created by Nintendo side, the data is stored on NintendoExtensinoBlock.
When extension data is not used, set the default flag as below. 
NINTENDO_EXT_BLOCK_NONE 0x00000000
u32 user_extension_block_size If an extension data is created by user side, the data is stored on UserExtensinoBlock.
When extension data is not used, set the default flag as below. 
USER_EXT_BLOCK_NONE     0x00000000
NintendoExtensionBlock If an extension data is created by Nintendo side, the data is stored on this NintendoExtensinoBlock.
UserExtensionBlock If an extension data is created by user side, the data is stored on this UserExtensinoBlock.
