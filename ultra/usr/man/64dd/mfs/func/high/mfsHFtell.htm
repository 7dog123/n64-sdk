mfsHFtell

Format: s32 mfsHFtell(MfsFile handle)

Arguments:

handle	File handle

Return value:File position (a negative value is an error)

Function: Obtains file position

Description: This function obtains the file position of the file specified by handle.

Main errors:

MFS_ERR_NO	Normal termination
MFS_ERR_FATAL	File open error
		The file is not opened.
