Functino name:mfsGetGameCode ────────────────────────────────────── Format: void mfsGetGameCode(u8* companyCode, u8* gameCode) Arguments: *companyCode Buffer for storing company code *gameCode Buffer for storing game code Return value:None Function: Obtains company code and game code Description: This function obtains the company code and game code that are set in the MFS library. To obtain only the company code or only the game code, specify NULL for the buffer corresponding to the code that is not to be obtained.
