Function name:mfsStrrChr ────────────────────────────────────── Format: u8* mfsStrrChr(const u8* src, s32 c) Arguments: src Character string to be searched c Search character Return value: src pointer where character c appeared last Function: Finds position where character c appears last Description: This function searches for the position within the character string src where the character c appears last. A pointer indicating the position within the character string src at which the character c appears last is returned for the return value. If the character c is not found, NULL is returned.
