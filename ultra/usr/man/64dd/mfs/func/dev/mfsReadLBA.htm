mfsReadLBA

Format

s32 mfsReadLBA(u32 startLBA, u32 nLBAs, void* buf)

Arguments

startLBA Starting block address
nLBAs Number of blocks
*buf Buffer

Return Value

Error

Function

Reads disk

Description

This function reads data from disk in units of LBAs. This is an mfsReadWriteLBA macro.
