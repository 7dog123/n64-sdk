mfsWriteLBA

Format

s32 mfsWriteLBA(u32 startLBA, u32 nLBAs, void* buf)

Arguments

startLBA Starting block address
nLBAs Number of blocks
*buf Buffer

Return value

Error

Function

Writes to disk

Description

This function writes data to disk in units of LBAs. This is an mfsReadWriteLBA macro.
