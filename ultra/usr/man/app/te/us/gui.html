  1. Menu
    Menu Image Explanation
    1. File
  2. New... opens the dialog for creating either a TMF file or a TPF file. Select either of these two file types, and the corresponding TPF/TMF dialog opens.
  3. Open... opens the dialog to open a file.
  4. Close closes the file that is currently being edited. If the file has not been saved at this time, a dialog opens asking the user whether or not to save the file before closing, or to cancel the command.
  5. Save saves the file that is currently being edited. If you act to save a new image which has not been named yet, the Save File dialog opens, giving you the opportunity to give the image a file name and then save.
  6. Save As... saves the file that is currently being edited under the user-specified name. The Save File dialog opens, giving you the opportunity to give the file a new a file name.
  7. Import imports another file into an image. See also: Importable files
  8. Export outputs a picture image to another file.
    See also: Exportable files
  9. Print, Print preview, and Printer settings are not supported at this time.
  10. Exit quits the application. All files currently being edited are closed. If some file has not been saved at this time, a dialog opens asking the user whether or not to save the file before closing, or to cancel the command.
    11. 2. Edit
  11. Undo lets you undo a change and return the image to its original form. The application records the state of the image back through the last 100 changes. See also: Undo
  12. Redo has the opposite effect of redoing what was just undone. See also: Redo
  13. Cut copies the selected image to the clipboard and then deletes it. See also: Cut
  14. Copy copies the selected image to the clipboard. See also: Copy
  15. Paste inserts the image that is in the clipboard. See also: Paste
  16. Flip horizontal - Flip vertical turns the selected image around. See also: Flip horizontal and Flip vertical
  17. Add to project adds the currently active TMF (must be attached to TLF) to the project.
    18. 3. Projects
  18. New... opens the New Project File dialog. Please select this item when editing a new TPF.
  19. Open... opens the Open File dialog.
  20. Close closes the project (TPF) that is currently being edited. If the project file has not been saved at this time, a dialog opens asking the user whether or not to save the file before closing, or to cancel the command.
  21. Save saves the project (TPF) that is currently being edited. If you act to save a new project which has not been named yet, the Save File dialog opens, giving you the opportunity to give the project a file name and then save.
  22. Save As... saves the project (TPF) that is currently being edited under the user-specified name. The Save File dialog opens, giving you the opportunity to give the project a new a file name.
  23. Add picture adds a new picture to the project that is currently being edited. The Picture Settings dialog opens for you to set and register the new picture name and the image and the palette. See also: Add
  24. Change picture changes the picture composition registered in the the currently edited project. The Picture Settings dialog opens for you to change the composition of the image and the palette. See also: Change
  25. Remove picture removes a picture registered in the the currently edited project. See also: Remove
  26. Properties opens the Properties of the currently edited project. See also: Directory
    27. 4. Palettes
  27. New... opens the New Palette File dialog. Please select this item when editing a new TLF.
  28. Open... opens the Open File dialog.
  29. Close closes the palette (TLF) that is currently being edited. If the palette file has not been saved at this time, a dialog opens asking the user whether or not to save the file before closing, or to cancel the command.
  30. Save saves the palette (TLF) that is currently being edited. If you act to save a new palette which has not been named yet, the Save File dialog opens, giving you the opportunity to give the project a file name and then save.
  31. Save As... saves the palette (TLF) that is currently being edited under the user-specified name. The Save File dialog opens, giving you the opportunity to give the project a new a file name.
  32. Edit edits the colors of the specified palette.
  33. Attach allocates the current palette to the currently active image. Nothing is displayed for an image that does not have an attached palette.
  34. Undo lets you undo a change and return the palette to its original form. The application records the last 100 changes. See also: Undo
  35. Redo has the opposite effect of redoing what was just undone. See also: Redo
  36. Copy copies the selected palette to the clipboard.
  37. Paste inserts the palette that is in the clipboard.
    38. 5. Draw
  38. Select selects a rectangular area for editing. See also: Select rectangle
  39. Zoom is the mode for viewing the selected screen position in an enlarged or reduced form. See also: Zoom function
  40. Pixel draws one point. The point is drawn with the color currently assigned to the pen. See also: Pixel
  41. Pen is the "free draw" mode. The line is drawn with the color currently assigned to the pen. See also: Pen
  42. Line draws a straight line. The line is drawn with with the color currently assigned to the pen. See also: Line
  43. Box draws a rectangle. The rectangle is drawn with with the color currently assigned to the pen. See also: Box
  44. Box fill draws a filled-in rectangle which is filled with the color currently assigned to the pen. See also: Box fill
  45. Circle draws a circular shape. The circle is drawn with the color currently assigned to the pen. See also: Circle
  46. Circle fill draws a filled-in circle which is filled with the color currently assigned to the pen. See also: Circle fill
  47. Paint fills the selected area with the color currently assigned to the pen. See also: Paint
  48. Pick with right-click picks up a color from the image and assigns it as the current color of the pen. See also: Pick with right-click
    49. 6. View
  49. Tool bar turns display of the Tool bar ON/OF.
  50. Status bar turns display of the Status bar ON/OFF.
  51. Project bar turns display of the Project dialog bar ON/OFF.
  52. Palette turns display of the Palette dialog bar ON/OFF.
  53. Draw mode bar turns display of the Draw tool bar ON/OFF.
  54. View bar turns display of the View tool bar ON/OFF.
  55. Zoom enlarges the image on display. See also: Zoom function
  56. Grid turns the grid display ON/OFF. See also: Grid
  57. Preview 1/2 turns the Preview function ON/OFF. See also: Preview function
    58. 7. Windows The same as in other applications. Explanation is omitted.
    8. Help

  59. About opens the About dialog. Use this to check which version of the product and which plug-ins have been installed.

    "(85,135) 200%" indicates the pen coordinates and the magnification.
    "(320,240) 8bit" indicates the size and the bits of the currently edited image.

    dialog bar    		 Pages increase when the palettes increase.
    The 4bit/8bit palette list is displayed, and the pen color can be selected with the mouse.
    You can change the color settings by double-clicking the mouse or pressing the "Edit" button to open the Color settings dialog.
