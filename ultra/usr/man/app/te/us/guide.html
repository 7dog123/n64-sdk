  1. Projects
    1. Creating a project
      1. Open the Project menu and select "New..." to open the Project page of the New File dialog.
      2. Enter the TPF file name/ TPF directory name inside the dialog.
          Press this button to open the Select Directory dialog.
          If no TMF/TLF directory is specified, then the directory becomes the same one as was specified for the TPF directory.
      3. When the OK button is pressed, any open projects are closed.
        If the project has been edited, a dialog opens asking whether or not to save the file.
      4. If the new project has been created correctly, it will be opened in an editable state.
    2. Opening a project
      1. Any project that is already open will be closed.
        If that project has been edited, a dialog will open asking whether or not to save the file.
      2. To open a project, open the Project menu and select "Open..." This opens the Open File dialog, where you specify the target TPF and then press the Open button.
      3. If there are any previously edited projects, then a list of the recently edited files will be displayed in the File menu, and it is also possible to select the file from here.
      4. If the file is properly loaded, the project will open in an editable state.
    3. Saving a project
      1. To save a project, you can select either "Save" or "Save As..." from the Project menu.
      2. When saving a file for the first time or with "Save As..." the Save File dialog will open. Here you can specify a name for the file.
    4. Adding a picture to a project
        Note: Pictures are assemblages of related images and palettes.
        Pictures reference TMF/TLF file names, so the TMF/TLF assemblage must be in the TMF/TLF directory.
      1. From the Project dialog bar or the Project menu, select "Picture" "Add" (or "Change" if manipulating an existing picture) to open the Edit Picture dialog.
      2. Select picture name and TMF/TLF. Since the TMF/TLF combo box lists the files that exist in the directory specified by the TMF/TLF directory, the selection can be made from here.
        To create a new TMF/TLF, press the New button.
        When TMF is 4bit and TLF is 8bit, you can select which palette number (0~15) to use for TLF.
      3. Open related pictures using the Project dialog bar or by clicking on the picture node in the project window.
      4. Delete the currently selected picture using the Project dialog bar or by selecting "Picture" "Delete" from the Project menu.
  2. Images
    1. Creating a new 4bit/8bit CI texture
      1. From the File menu, open the Image page of the New dialog.
      2. Select the size and bit mode, then press OK to open a new Texture Image window.
    2. Opening a 4bit/8bit CI Texture
      1. Open a texture image file from the File menu by selecting either "New..." or by selecting a file from the list of recently edited TMF files.
    3. Saving a 4bit/8bit CI texture
      1. Select either "Save" or "Save As" from File menu.
    4. Editing an existing image as a 4bit/8bit CI texture
      1. Do this using the Import function.
      2. First, create or open a texture image for editing.
      3. Then, select the target file format from "Import" in the File menu.
      4. Once imported, specify where to position the image.
          Press the Return key to set, or use the Escape key to cancel. Manipulate the tracker (selection frame)to enlarge or reduce.
      5. Alternately, a BMP file converted to 4bit/8bit can be directly opened from the File menu for TMF/TLF.
    5. Previewing an image during editing on the target (NINTENDO64) using IS-VIEWER64
      1. Select "Preview" in the View menu or press the NINTENDO64 button on the View tool bar to toggle preview to IS-VIEWER64 ON/OFF in each texture image window.
      2. When this check is ON, any changes to images or palettes will automatically update the texture on IS-VIEWER64.
    6. Exporting a created CI texture to a project
      1. Use the Export function to save the file as C language source.
      2. From the File menu, select "Export" and then select "C."
      3. In addition to specifying the header file and C source file name, you can specify extern (can be externally referenced)/static (internal static binding), turn const (change prohibited) ON/OFF, and specify 64bit alignment.
      4. With export, the C source file is created.
          If static is selected, the header file is not created.
  3. Palettes
    1. Creating a palette
      1. Open the Palette menu and select "New..." to open the Palette page of the New dialog.
      2. Select the bit mode and 16bit RGBA or 16bit IA and then press OK.
      3. If done correctly, a new page is created in the Palette dialog bar.
    2. Editing a palette
      1. In the initial state, the palette is completely black, so add color by pressing the "Properties" button or double-clicking to open the Color Selection dialog.
      2. To borrow from another palette, open the page for the other palette, select the range and press the Copy button, select the page and position where you want to begin pasting, and then press the Paste button.
    3. Saving a palette
      1. Open the Palette menu and select either "Save" or "Save As..."
