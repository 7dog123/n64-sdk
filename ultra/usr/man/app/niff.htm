NIFF 2.0

3hedra.niff

Ambient color and diffuse color from NIFF 2.0 materials are used. For example, in the case of the center object, the ambient color is green and the diffuse color is red.

hilight_donut.niff

This is a sample of specular highlight data. The luster of the object is expressed using the highlight texture.

LMS-lod.niff

This is a sample of LOD data. The object switches from L -> M -> S as it gets farther away.

texanim_star.niff

This is a sample of texture animation.

env_animal.niff

This is a sample using IK and Envelope. The shape changes according to the curved condition of the IK. If you wish to know the relationship between the positions of the Joint and the Effector nodes, switch the ICON menu in the Previewer to ON.

kirikobu.niff

This is a sample using IK and Envelope. This uses an Up Vector Constraint on the Joint node to express more complex movement than env_animal.niff.
