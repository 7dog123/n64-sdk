Default Settings

This section describes the file extensions that have default associations.

  • *.n64
    This is the extension for N64 files.
    When an n64 file is specified, it is transferred to and loaded on the specified target. There is no need to execute makemask in advance. Note: osSyncPrintf is not supported.
    Never change this setting.

  • *.bmp , *.rgb , *.sgi , *.tpf
    These are graphics files.
        *.bmp = Bitmap file
        *.rgb , *.sgi = SGI image file
        *.tpf = TextureEditor output file
        Converter: autoview.exe
        Target program: texv.n64

  • *.wav , *.aiff , *.aif
    These are sound files.
        *.wav = Microsoft RIFF wave
        *.aiff , *.aif = Audio Interchange File Format
        Converter: none
        Target program: Player.n64

  • *.bin , *.bfx
    These are files for N64 Sound Tools.
        *.bin = Song data file
        *.bfx = Sound effect data file
        Converter: stconv.exe
        Target program: soundtools.n64
