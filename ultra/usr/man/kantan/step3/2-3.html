3.2.3 N64 Functions to Draw 3D Shapes

Use the following N64 functions to draw three-dimensional objects:

  • Call gSPVertex to load the vertex data
  • Call gSPModifyVertex to change the vertex data
  • Call gSP1Triangle to draw one triangle
  • Call gSP2Triangles to draw two triangles
  • Call gSPSetGeometryMode G_CULL_BACK to turn on back face culling
  • Call gSPClearGeometryMode G_CULL_BACK to turn off back face culling
