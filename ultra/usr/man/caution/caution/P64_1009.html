10-9 Optional Delete for Game Notes (Required)

Caution
Ensure that unnecessary game notes can be deleted by the user, using the Controller Pak menu.

Discussion
If unnecessary game notes cannot be deleted by the user, the game may not be able to secure enough space for saving necessary data. Ensure that game notes created by another company's game can be deleted.
