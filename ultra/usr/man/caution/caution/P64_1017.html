10-17 Controller Pak Menu Library (Required)

Caution
Use the N64 library "libnos_jpn.a" for all games to be marketed in Japan and use "libnos.a" for all other markets.

Discussion
The N64 library "libnos_jpn.a" allows Japanese characters to be printed properly on the screen. If "libnos.a" is used for a game which is distributed in Japan, a note name which uses Japanese characters cannot be read by the user because Japanese characters will not be shown properly on the screen.
