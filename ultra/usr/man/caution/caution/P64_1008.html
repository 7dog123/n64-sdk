10-8 Controller Pak Menu - Contents (Required)

Caution
When the Controller Pak menu starts, ensure that a list of note numbers (1~16), note names, note extensions, number of pages used, and number of blank pages are displayed.

Discussion
This is for standardization of Controller Pak operation. The note extension must be used, even if the game does not use the multi-note function.
