10-3 Controller Pak Free Space (Required)

Caution
Before game play begins, an appropriate message must be displayed on the screen when there is not enough space left in the Controller Pak to save the desired data.

Discussion
Adhering to this caution will give the user the option to power-down and insert a different Controller Pak if there is not enough space left in the existing one. A maximum of 16 notes and a maximum of 123 pages can be created in one 256 Kbit Controller Pak.
