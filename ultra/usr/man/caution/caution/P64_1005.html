10-5 Damaged Controller Pak - General (Required)

Caution
When a damaged Controller Pak is inserted, the message "Damaged Controller Pak. Game cannot be saved." must be displayed on the screen.

Discussion
Use osPfsInitPak() for Controller Pak initialization. If an error is detected, use osPfsRepairId() to repair the Controller Pak. A message should be displayed, to give the user the option to clean the Controller Pak's contacts, before repairing the Controller Pak; as all existing data will be lost.
