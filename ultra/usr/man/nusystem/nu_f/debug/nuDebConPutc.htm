Function
nuDebConPutc

Syntax

void nuDebConPutc(u32 wndNo, u32 c)

Arguments

  • wndNo is the window number
    NU_DEB_CON_WINDOW0   Window 0
    NU_DEB_CON_WINDOW1   Window 1
    NU_DEB_CON_WINDOW2   Window 2
    NU_DEB_CON_WINDOW3   Window 3

  • c is the character

Return Value
None

Explanation
Outputs one character to console window.

The character "c" is output to the console window specified by wndNo. "c" is output as the low-order 8 bit in ACSII code.
