Function

nuContPakMgrInit
Syntax:
void nuContPakMgrInit(void)
Arguments:
None
Return Value:
None
Explanation:

Registers the Controller Pak Manager

Registers the Controller Pak Manager in the SI Manager. This function is called by the nuContInit function, so in most cases it does not need to be called. To remove the registration, use the nuContPakMgrRemove function.

For example:



	/* In this example, the Controller Manager and the Controller	*/
	/* Pak Manager are used without using the nuContInit function	*/
	/* Unlike with nuContInit function, the Rumble Pak cannot be used	*/
	void appContInit(void)
	{
		/* Initialize and activate the Controller Manager */
		nuContMgrInit();
		
		/* Activate the Controller Pak Manager */
		nuContPakMgrInit();
	}
Reference: nuSiMgrInit,nuContInit,nuContPakMgrRemove
