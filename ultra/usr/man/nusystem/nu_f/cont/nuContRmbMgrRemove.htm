Function

nuContRmbMgrRemove
Syntax:
void nuContRmbMgrRemove(void)
Arguments:

None

Return Value:

None

Explanation:

Removes the Rumble Pak Manager

This function removes the Rumble Pak Manager from the SI Manager. To register the Rumble Pak Manager, use the nuContRmbMgrInit function.

Reference:	nuSiMgrInit, nuContRmbMgrInit
