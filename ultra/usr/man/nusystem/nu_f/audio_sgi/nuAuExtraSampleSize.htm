Global Variables

nuAuExtraSampleSize
Syntax
s16 nuAuExtraSampleSize

Explanation
Extra sample size

Set the adjusted size of the audio data which are created at one time. This is normally 0 (zero).

When this value is changed, set the value before calling the following functions:

nuAuInit
nuAuInitEx
nuAuMgrInit
nuAuMgrInitEx
