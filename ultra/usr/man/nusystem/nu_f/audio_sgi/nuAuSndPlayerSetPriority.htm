Function
nuAuSndPlayerSetPriority (Macro)
Syntax
nuAuSndPlayerSetPriority(ALSndId id,u8 priority)

Arguments

  • id is the Sound ID

  • priority is the Sound Priority (0-127)

Return Value
None

Explanation
Set the priority of the sound object for the sound player.

A value of 0 is the lowest priority and a value of 127 is the highest priority.
