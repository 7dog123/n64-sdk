Function
nuAuSeqPlayerSetVol (Macro)
Syntax
void nuAuSeqPlayerSetVol(u32 player_no, s16 vol)

Arguments

  • player_no is the sequence player number

  • vol is the set volume (0 - 0x7fff)

Return Value
None

Explanation
Set the volume for the whole sequence for the player which is specified by sequence player number. 0x7fff indicates that the volume is full and 0 indicates it is off.
