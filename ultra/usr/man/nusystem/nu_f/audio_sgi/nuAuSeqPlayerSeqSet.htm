Function
nuAuSeqPlayerSeqSet
Syntax 
void nuAuSeqPlayerSeqSet(u8* seq_addr)

Arguments

  • seq_addr is the ROM address for MIDI sequence bank data

Return Value
None

Explanation
Set ROM address for MIDI sequence bank data in the sequence player. MIDI sequence bank is MIDI sequence data which is combined into one file by the sbc tool.
