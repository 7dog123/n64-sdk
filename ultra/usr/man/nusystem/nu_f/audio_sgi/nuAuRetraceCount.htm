Global Variables

nuAuRetraceCount
Syntax
u8 nuAuRetraceCount

Explanation
Audio processing unit.

This is the size of audio data created at one time. When 1 is specified, one frame of audio data is created. Normally, this is set to the same value as the retrace period set in the scheduler.
The default value is 1.

When this variable is changed, set the value before calling the following functions:

nuAuInit
nuAuInitEx
nuAuMgrInit
nuAuMgrInitEx
