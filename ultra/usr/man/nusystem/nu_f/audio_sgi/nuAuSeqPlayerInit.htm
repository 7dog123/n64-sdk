Function
nuAuSeqPlayerInit
Syntax 
void nuAuSeqPlayerInit(ALSeqpConfig* seqpconfig,u32 size,u32 player_no)

Arguments

  • seqconfig is the structure of the sequence player

  • size is the buffer size for sequence data storage

  • player_no is the number of player to be initialized (0,1)

Return Value
None

Explanation
Initialize the sequence player.
