Function
nuAuSeqPlayerGetChlProgram (Macro)
Syntax
s32 nuAuSeqPlayerGetChlProgram(u32 player_no, u8 chan)

Arguments

  • player_no is the sequence player number

  • chan is the MIDI channel

Return Value
Program number

Explanation
Obtain program number for the MIDI channel specified by the player which is specified by the sequence player number. Return -1 if a program number is not allocated.
