Function
nuAuSeqPlayerSetTempo (Macro)
Syntax
void nuAuSeqPlayerSetTempo(u32 player_no, S32 tempo)

Arguments

  • player_no is the sequence player number

  • tempo is the tempo (in micro seconds/quarter note

Return Value
None

Explanation
Set tempo for sequence for player which is specified by sequence player number.
