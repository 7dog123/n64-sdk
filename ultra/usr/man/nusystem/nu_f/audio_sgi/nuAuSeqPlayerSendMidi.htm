Function
nuAuSeqPlayerSendMidi (Macro)
Syntax 
void nuAuSeqPlayerSendMidi(u32 player_no, long ticks,u8 status, u8 byte1, u8 byte2)

Arguments

  • player_no is the sequence player number

  • ticks is the timing offset by MIDI clock ticks when MIDI events occur

  • status is the MIDI status byte of message

  • byte1 is the first byte of message

  • byte2 is the second byte of message

Return Value
None

Explanation
Send MIDI message to the player which is specified by sequence player number. Please refer to the N64 Man Page "alSeqpSendMidi" for details.
