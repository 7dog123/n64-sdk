Function
nuAuSndPlayerSetSound (Macro)
Syntax
void nuAuSndPlayerSetSound(ALSndId SndId)

Arguments

  • SndId is the ID for sound to be played back

Return Value
None

Explanation
Specify the sound object for sound player.

This is used when specifying the sound for which stop or effect will be applied by using the sound ID which is returned by nuAuSndPlayerPlay().
