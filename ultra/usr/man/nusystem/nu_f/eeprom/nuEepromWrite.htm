Function

nuEepromWrite

Syntax

s32 nuEepromWrite(u8 address, u8* buffer, u16 len)

Arguments

  • address is the EEPROM address where data is to be written

  • *buffer is the pointer to the buffer where the data is stored

  • len is the data length

Return Value

  • 0     Normal termination<

  • -1   Address is incorrect, or the EEPROM Manager is not registered

  • CONT_NO_RESPONSE_ERROR   The interface does not respond

Explanation
Writes data to EEPROM

Writes data from the specified "buffer" to the specified "address" in EEPROM. The specified "address" is 0-63 for 4K EEPROM and 0-255 for 16K EEPROM, and len*8 bytes of data are read.
