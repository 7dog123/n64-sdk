Function
nuAuStlSndPlayerSetSndPan(Macro)
Syntax:
s32 nuAuStlSndPlayerSetSndPan(u32 handle, s32 pan)
Arguments:
handle	sound handle
pan	pan scale ( 0 - 0x100 )
Return Value:
Number of channels for which command was executed
Explanation:

Set pan position of the sound which is specified by the sound handle.
