Function
nuAuStlPtrBankSet

Syntax:

void nuAuStlPtrBankSet(u8 *pbk_addr, u32 pbk_size, u8 *wbk_addr)

Arguments: 

pbk_addr        ROM address of sample pointer bank
pbk_size        Size of sample pointer bank
wbk_addr        ROM address of sample wave bank

Return Value: 

None

Description:

Reads and initializes the sample pointer bank. Then makes associations with the sample wave bank and registers it in the mus library.
