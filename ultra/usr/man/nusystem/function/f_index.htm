Memory Allocation Routines

InitHeap Reserves and initializes a memory allocation area
malloc Reserves a memory block that is greater in size than the area that was reserved by InitHeap function
free Frees the memory block that was reserved by the InitHeap function
realloc Resizes memory block in an area reserved by the InitHeap function
calloc Reserves memory block with a specified number of elements of a specified size bytes each in the area reserved by the InitHeap function
memalign Reserves memory block in area reserved by InitHeap function from area for which alignment was adjusted

For Debugging:

malloc_memcheck Checks the memory allocation area
malloc_memdsp Makes 60 digits into 1 line for the specified one-dimensional character string
