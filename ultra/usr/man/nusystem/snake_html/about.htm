About Snake Tail Hack Snake Tail Hack is described below.

Game Contents

4-player game

A player wins by defeating all the snake heads except itself by manipulating the snake head.

If the game is not finished within a certain period of time, the player with the longest tail is the winner.

Also, when all the tails are taken, the game is over.

When 4 Controller Paks are inserted, each player's result can be saved and replayed using the replay mode.

Snake Head

Each player controls one snake head.

By adding more tail segments, the snake head becomes stronger. The initial number of tail segments is 5.
