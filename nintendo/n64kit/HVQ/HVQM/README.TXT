HVQM2                                      1999/8/4


This package is for using the HVQM motion picture encoding method with the 
Nintendo 64.

The following programs are stored in each directory:

hvqm2enc       Still image compression tool (version 1.3)
libhvqm2       N64-HVQ2 library (version 1.2)
sample         Sample program (version 1.2)
nuhvqm         NuSystem version of sample program (version 1.0)
