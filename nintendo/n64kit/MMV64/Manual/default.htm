Default Settings

The extensions which are registered as defaults are explained here.

  • *.n64
    As you know, this is an n64 file.
    When an n64 file is specified, it is transferred to and executed on the target.
    There is no need to execute makemask in advance.
    Please also note that osSyncPrintf is not supported.
    * You absolutely must not change this setting.

  • *.bmp , *.rgb , *.sgi , *.tpf
    These are graphics files.
        *.bmp ... Bitmap file
        *.rgb , *.sgi ... SGI image file
        *.tpf ... TextureEditor output file
        Converter : autoview.exe
        Target program : texv.n64

  • *.wav , *.aiff , *.aif
    These are sound files.
        *.wav ... Microsoft RIFF wave
        *.aiff , *.aif ... Audio Interchange File Format
        Converter : none
        Target program : Player.n64

  • *.bin , *.bfx
    These are files for N64 Sound Tools.
        *.bin ... Song data file
        *.bfx ... Sound effect data file
        Converter : stconv.exe
        Target program : soundtools.n64

  • *.hvq , *.hvqm
    compressed image file
        *.hvq ... Stationary HVQ Compressed File
        *.hvqm ... Animated HVQM Compressed File
        Converter: None
        Target Program:hvq.n64, hvqm.n64
    * Also possible to register HVQ Encoder in converter
